# Gold Digger

This is a solo project that focuses on backend development, REST APIs, file handling, and client-server communication in a full stack JavaScript application. The purpose of this mini project was to create a simulated gold investment platform where users can view live gold prices, invest a custom amount, and instantly see how many ounces of gold they would receive based on the current market price. The application also records each transaction on the server to simulate purchase tracking.

This project idea came from Scrimba's Fullstack Developer Path, where students complete solo projects after finishing different modules and lessons. The goal of this project was to strengthen my understanding of how web servers work, how API endpoints are created, and how frontend applications communicate with backend services. Instead of using Express, I built the server using Node.js' built-in HTTP module, which gave me a better understanding of routing, request handling, and serving static files.

## Getting Started

1. Clone the repository.

```bash
git clone <repository-url>
```

2. Navigate to the project folder.

```bash
cd gold-digger
```

3. Install dependencies.

```bash
npm install
```

4. Start the server.

```bash
npm start
```

5. Open your browser and visit:

```text
http://localhost:8000
```
